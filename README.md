# KernelKit

封装 iOS/Macosx 常用的底层API，涉及动态库，线程，内存，Crash, Hook, I/O, ObjC-ABI等内容，持续更新

## Features
1. dyld
2. memory
3. I/O
    * mmap
4. thread
5. crash
    * signal
    * NSException
    * c++ termination
6. hook
    * swizzle
    * fishhook
7. ObjC-ABI
    * block
    * object
    * autorelease pool page
